# MyApp项目模块说明

app：自定义进度框

说明：进度对话框透明背景

mycamera：前置摄像头拍照

知识点：调用手机前置摄像头，并截取图片展示到手机界面

highlightSearch：关键字高亮显示搜索
知识点：
1、读取xml文件数据

2、根据EditText输入关键字匹配ListView选项，将与关键字匹配项显示出来

3、重写BaseAdapter，实时动态更新ListView选项
